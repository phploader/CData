# **9. Cookbook – Practical Examples**

This chapter offers a collection of practical examples that cover typical CData use cases.
All examples are immediately usable and show how `$D`, `$F`, `get_object()`, and `set_object()` are used in real projects.

---

# **9.1 Load all articles**

```php
$F['ARTICLE'] = [];
$db->get_object($D, $F);
```

Result:

```php
$D['ARTICLE']['D'][ID][fieldname]
```

---

# **9.2 Load only active articles**

```php
$F['ARTICLE']['W'][0]['Active'] = 1;
$db->get_object($D, $F);
```

---

# **9.3 Filter articles by name**

```php
$F['ARTICLE']['W'][0]['Name']['LIKE'] = '%Test%';
$db->get_object($D, $F);
```

---

# **9.4 Load articles by multiple IDs**

```php
$F['ARTICLE']['W'][0]['ID'] = ['A1', 'A2', 'A3'];
$db->get_object($D, $F);
```

---

# **9.5 Sort articles**

```php
$F['ARTICLE']['O'][0]['Name'] = 'ASC';
$db->get_object($D, $F);
```

---

# **9.6 Paginate articles**

```php
$F['ARTICLE']['L']['STEP']  = 20;
$F['ARTICLE']['L']['START'] = 0;
$db->get_object($D, $F);
```

---

# **9.7 Load articles + attributes**

```php
$F['ARTICLE']['ATTRIBUTE'] = [];
$db->get_object($D, $F);
```

---

# **9.8 Filter articles that have a specific attribute**

```php
$F['ARTICLE']['W'][0]['ATTRIBUTE']['W'][0]['Name'] = 'Color';
$db->get_object($D, $F);
```

---

# **9.9 Load all articles, but filter attributes**

```php
$F['ARTICLE']['ATTRIBUTE']['W'][0]['Name'] = 'Color';
$db->get_object($D, $F);
```

---

# **9.10 Load articles + attributes + language**

```php
$F['ARTICLE']['ATTRIBUTE']['LANGUAGE'] = [];
$db->get_object($D, $F);
```

---

# **9.11 Create an article**

```php
$D['ARTICLE']['D']['A100']['Active'] = 1;
$D['ARTICLE']['D']['A100']['Name']   = 'New Article';

$db->set_object($D);
```

---

# **9.12 Update an article**

```php
$D['ARTICLE']['D']['A100']['Name'] = 'Updated Article';
$db->set_object($D);
```

---

# **9.13 Delete an article**

```php
$D['ARTICLE']['D']['A100']['__DELETE__'] = 1;
$db->set_object($D);
```

---

# **9.14 Add an attribute to an article**

```php
$D['ARTICLE']['D']['A100']['ATTRIBUTE']['D']['Color']['Name'] = 'Red';
$db->set_object($D);
```

---

# **9.15 Delete an article attribute**

```php
$D['ARTICLE']['D']['A100']['ATTRIBUTE']['Color']['__DELETE__'] = 1;
$db->set_object($D);
```

---

# **9.16 Delete all attributes of an article**

```php
$D['ARTICLE']['D']['A100']['ATTRIBUTE']['__DELETE__'] = 1;
$db->set_object($D);
```

---

# **9.17 Save language data**

```php
$D['ARTICLE']['D']['A100']['LANGUAGE']['DE']['Title'] = 'Title DE';
$D['ARTICLE']['D']['A100']['LANGUAGE']['EN']['Title'] = 'Title EN';

$db->set_object($D);
```

---

# **9.18 Save a recursive structure**

```php
$D['ARTICLE']['D']['A100']['ATTRIBUTE']['D']['Color']['PROPERTY']['D']['HEX']['Value'] = '#FF0000';
$db->set_object($D);
```

---

# **9.19 Load a recursive structure**

```php
$F['ARTICLE']['ATTRIBUTE']['PROPERTY']['LANGUAGE']['META'] = [];
$db->get_object($D, $F);
```

---

# **9.20 Save multiple tables simultaneously**

```php
$D['ARTICLE']   = [...];
$D['ATTRIBUTE'] = [...];
$D['STOCK']     = [...];

$db->set_object($D);
```

---

# **Summary**

This cookbook demonstrates common use cases:

- loading data
- filtering data
- sorting & paging
- loading CHILD data
- processing recursive structures
- saving, updating and deleting data
- handling multiple tables at once

These examples serve as templates for your own implementations and can be extended arbitrarily.
