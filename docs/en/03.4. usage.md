# **3.4 Comparison Operators**

```php
$F['ARTICLE']['W'][0]['Price']['>']  = 10;
$F['ARTICLE']['W'][0]['Price']['<']  = 100;
$F['ARTICLE']['W'][0]['Active']['!='] = 0;
```

---

# **3.5 LIKE Variants**

```php
$F['ARTICLE']['W'][0]['Name']['LIKE']   = '%test%';   // contains
$F['ARTICLE']['W'][0]['Name']['LIKE%%'] = 'test';     // contains (short form)
$F['ARTICLE']['W'][0]['Name']['LIKE%-'] = 'test';     // ends with
$F['ARTICLE']['W'][0]['Name']['LIKE-%'] = 'test';     // starts with
```

---

# **3.6 IN / NOT IN**

```php
$F['ARTICLE']['W'][0]['ID'] = ['1','2','3'];          // IN
$F['ARTICLE']['W'][0]['ID']['NOTIN'] = ['1','2','3']; // NOT IN
```

---

# **3.7 Sorting – ORDER (O)**

```php
$F['ARTICLE']['O'][0]['Name'] = 'ASC';
```

# **3.8 Limit & Offset – LIMIT (L)**

```php
$F['ARTICLE']['L']['STEP']  = 10; // number of records
$F['ARTICLE']['L']['START'] = 0;  // offset
```
