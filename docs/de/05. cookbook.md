# **9. Cookbook – Praktische Beispiele**

Dieses Kapitel bietet eine Sammlung von praxisnahen Beispielen („Cookbook“), die typische Anwendungsfälle mit CData abdecken.\
Alle Beispiele sind sofort nutzbar und zeigen, wie `$D`, `$F`, `get_object()` und `set_object()` in realen Projekten eingesetzt werden.

---

# **9.1 Alle Artikel laden**

```php
$F['ARTICLE'] = [];
$db->get_object($D, $F);
```

Ergebnis:

```php
$D['ARTICLE']['D'][ID][Feldname]
```

---

# **9.2 Nur aktive Artikel laden**

```php
$F['ARTICLE']['W'][0]['Active'] = 1;
$db->get_object($D, $F);
```

---

# **9.3 Artikel nach Name filtern**

```php
$F['ARTICLE']['W'][0]['Name']['LIKE'] = '%Test%';
$db->get_object($D, $F);
```

---

# **9.4 Artikel nach mehreren IDs laden**

```php
$F['ARTICLE']['W'][0]['ID'] = ['A1', 'A2', 'A3'];
$db->get_object($D, $F);
```

---

# **9.5 Artikel sortieren**

```php
$F['ARTICLE']['O'][0]['Name'] = 'ASC';
$db->get_object($D, $F);
```

---

# **9.6 Artikel paginieren (Paging)**

```php
$F['ARTICLE']['L']['STEP']  = 20;
$F['ARTICLE']['L']['START'] = 0;
$db->get_object($D, $F);
```

---

# **9.7 Artikel + Attribute laden**

```php
$F['ARTICLE']['ATTRIBUTE'] = [];
$db->get_object($D, $F);
```

---

# **9.8 Artikel filtern, die ein bestimmtes Attribut besitzen**

```php
$F['ARTICLE']['W'][0]['ATTRIBUTE']['W'][0]['Name'] = 'Farbe';
$db->get_object($D, $F);
```

---

# **9.9 Alle Artikel laden, aber Attribute filtern**

```php
$F['ARTICLE']['ATTRIBUTE']['W'][0]['Name'] = 'Farbe';
$db->get_object($D, $F);
```

---

# **9.10 Artikel + Attribute + Language laden**

```php
$F['ARTICLE']['ATTRIBUTE']['LANGUAGE'] = [];
$db->get_object($D, $F);
```

---

# **9.11 Artikel anlegen**

```php
$D['ARTICLE']['D']['A100']['Active'] = 1;
$D['ARTICLE']['D']['A100']['Name']   = 'Neuer Artikel';

$db->set_object($D);
```

---

# **9.12 Artikel aktualisieren**

```php
$D['ARTICLE']['D']['A100']['Name'] = 'Artikel aktualisiert';
$db->set_object($D);
```

---

# **9.13 Artikel löschen**

```php
$D['ARTICLE']['D']['A100']['__DELETE__'] = 1;
$db->set_object($D);
```

---

# **9.14 Attribut zu einem Artikel hinzufügen**

```php
$D['ARTICLE']['D']['A100']['ATTRIBUTE']['D']['Color']['Name'] = 'Rot';
$db->set_object($D);
```

---

# **9.15 Attribut eines Artikels löschen**

```php
$D['ARTICLE']['D']['A100']['ATTRIBUTE']['Color']['__DELETE__'] = 1;
$db->set_object($D);
```

---

# **9.16 Alle Attribute eines Artikels löschen**

```php
$D['ARTICLE']['D']['A100']['ATTRIBUTE']['__DELETE__'] = 1;
$db->set_object($D);
```

---

# **9.17 Language‑Daten speichern**

```php
$D['ARTICLE']['D']['A100']['LANGUAGE']['DE']['Title'] = 'Titel DE';
$D['ARTICLE']['D']['A100']['LANGUAGE']['EN']['Title'] = 'Title EN';

$db->set_object($D);
```

---

# **9.18 Rekursive Struktur speichern**

```php
$D['ARTICLE']['D']['A100']['ATTRIBUTE']['D']['Color']['PROPERTY']['D']['HEX']['Value'] = '#FF0000';
$db->set_object($D);
```

---

# **9.19 Rekursive Struktur laden**

```php
$F['ARTICLE']['ATTRIBUTE']['PROPERTY']['LANGUAGE']['META'] = [];
$db->set_object($D, $F);
```

---

# **9.20 Mehrere Tabellen gleichzeitig speichern**

```php
$D['ARTICLE']   = [...];
$D['ATTRIBUTE'] = [...];
$D['STOCK']     = [...];

$db->set_object($D);
```

---

# **Zusammenfassung**

Dieses Cookbook zeigt typische Anwendungsfälle:

- Daten laden
- Daten filtern
- Sortieren & paginieren
- CHILD‑Daten laden
- Rekursive Strukturen verarbeiten
- Daten speichern, aktualisieren und löschen
- Mehrere Tabellen gleichzeitig verarbeiten

Diese Beispiele dienen als Vorlage für eigene Implementierungen und können beliebig erweitert werden.
