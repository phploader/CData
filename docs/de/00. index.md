# **Willkommen zur CData‑Dokumentation**

CData ist ein leichtgewichtiges, aber äußerst flexibles Datenmanagement‑System für PHP.\
Es ermöglicht das Arbeiten mit komplexen, rekursiven Datenstrukturen – ganz ohne SQL‑Abfragen, ORM‑Modelle oder starre Datenbankschemata. Stattdessen basiert CData vollständig auf zwei zentralen Konzepten:

- **`$D`** – der Datencontainer für Lesen und Schreiben
- **`$F`** – der Filtercontainer für Abfragen

Die gesamte Datenbankstruktur wird über ein **Pattern‑System** definiert, das Tabellen, Felder, Beziehungen und CHILD‑Strukturen beschreibt.\
CData erzeugt daraus automatisch die notwendigen SQLite‑Tabellen und verwaltet Insert, Update und Delete selbstständig.

Diese Wiki ergänzt das README und bietet eine ausführliche, praxisorientierte Dokumentation mit vielen Beispielen, Best Practices und Erklärungen zur internen Funktionsweise.

## **Für wen ist CData gedacht?**

CData richtet sich an Entwickler, die:

- flexible Datenstrukturen benötigen
- ohne SQL arbeiten möchten
- komplexe Beziehungen ohne ORM‑Overhead abbilden wollen
- ein System bevorzugen, das Daten und Struktur in einem einzigen Array verwaltet
- modulare, erweiterbare Anwendungen entwickeln

Ob Shopsystem, CMS, Konfigurator oder Datenverwaltung – CData eignet sich für alle Projekte, die dynamische Strukturen benötigen.
